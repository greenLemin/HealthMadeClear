# src/components/dashboard/

## Responsibility

Dashboard widgets for authenticated users.

## Key Files

- `AchievementCard.tsx`: Achievement display card with icon, title, earned status
- `DashboardSidebar.tsx`: Dashboard sidebar navigation

## Subdirectories

- `skeletons/`: Loading skeleton components for dashboard sections

## Integration

- Consumed by: `/dashboard/*` pages
- Depends on: Dashboard data from `src/lib/dashboard.ts`
