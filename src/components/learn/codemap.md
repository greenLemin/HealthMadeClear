# src/components/learn/

## Responsibility

Learning catalog card components.

## Key Files

- `LessonCard.tsx`: Lesson preview card — title, description, category, difficulty, duration, image
- `LearningPathCard.tsx`: Learning path card — title, description, lesson count, difficulty

## Integration

- Consumed by: `/learn`, `/learning-paths`, home page, path detail pages
- Depends on: `LessonCategoryId` types, category visual mappings
