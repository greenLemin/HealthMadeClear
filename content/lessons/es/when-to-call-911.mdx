## Matriz de decisiones

| Situación                                              | Dónde ir               | Por qué                                        |
| ------------------------------------------------------ | ---------------------- | ---------------------------------------------- |
| Dolor en el pecho, dificultad para respirar            | **Llama al 911**       | Emergencia que necesita tratamiento en minutos |
| Hueso roto (sin hueso expuesto)                        | **Urgencia o ER**      |                                                |
| Fiebre 101°F+, se siente mal pero puede caminar/hablar | **Urgencia**           |                                                |
| Fiebre 103°F+ o confusión                              | **ER**                 | Podría ser infección grave                     |
| Corte profundo con sangrado                            | **ER**                 | Puede necesitar puntos                         |
| Resfriado leve, chequeo de rutina                      | **Consultorio médico** |                                                |

## Llama al 911 para emergencias

### Señales de ataque cardíaco

- Molestia en el pecho (presión, opresión)
- Dolor que se extiende a brazos, espalda, cuello, mandíbula
- Falta de aire
- Sudor frío, náuseas, mareos

### Señales de accidente cerebrovascular (RÁPIDO)

- **R**ostro — ¿un lado del rostro se cae?
- **Á**mbos brazos — ¿un brazo se desploma?
- **P**alabra — ¿arrastra las palabras?
- **I**nmediatamente — llama al 911
- **O**portunidad — el tiempo es crítico

### Otras emergencias

- Dificultad para respirar
- Reacción alérgica grave (hinchazón de cara/labios, dificultad para respirar)
- Sangrado grave
- Convulsión
- Lesión en la cabeza con confusión o desmayo
- Envenenamiento o sobredosis
- Quemadura grave

:::warning
Si no estás seguro de si es una emergencia, **llama al 911 de todos modos.** Los despachadores pueden ayudar a evaluar.
:::
