## Por qué varían los programas de detección

Las recomendaciones de detección dependen de:

- Edad
- Sexo asignado al nacer
- Antecedentes familiares
- Historial médico personal
- Factores de estilo de vida (tabaquismo, peso, etc.)

## Edades 18-39

| Prueba             | Frecuencia    | Notas                                  |
| ------------------ | ------------- | -------------------------------------- |
| Presión arterial   | Cada 1-2 años | Más seguido si alta                    |
| Colesterol         | Cada 4-6 años | Desde los 20 si hay factores de riesgo |
| Examen dental      | Cada 6 meses  |                                        |
| Examen de la vista | Cada 2-4 años |                                        |

## Edades 40-49

| Prueba               | Frecuencia       | Notas                         |
| -------------------- | ---------------- | ----------------------------- |
| Presión arterial     | Cada año         |                               |
| Colesterol           | Cada 4-6 años    |                               |
| Diabetes             | Cada 3 años      | Desde los 35 si hay sobrepeso |
| Cáncer colorrectal   | **Desde los 45** | Colonoscopía cada 10 años     |
| Mamografía (mujeres) | Cada 1-2 años    |                               |
| Examen dental        | Cada 6 meses     |                               |

## Edades 50-64

| Prueba               | Frecuencia    | Notas          |
| -------------------- | ------------- | -------------- |
| Presión arterial     | Cada año      |                |
| Colesterol           | Cada 4-6 años |                |
| Diabetes             | Cada 3 años   |                |
| Cáncer colorrectal   | Cada 10 años  |                |
| Mamografía (mujeres) | Cada 1-2 años |                |
| Vacuna herpes zóster | Dos dosis     | A partir de 50 |

## Edades 65+

| Prueba             | Frecuencia    | Notas |
| ------------------ | ------------- | ----- |
| Presión arterial   | Cada año      |       |
| Colesterol         | Cada 4-6 años |       |
| Cáncer colorrectal | Hasta 75      |       |
| Osteoporosis       | Cada 2 años   |       |
| Vacuna neumonía    | 1-2 dosis     |       |
| Vacuna gripe       | Cada año      |       |

:::success
Pon recordatorios de detección en tu calendario. Trátalos como cualquier cita importante.
:::
