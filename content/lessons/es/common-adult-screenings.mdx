## Por qué varían los calendarios de detección

Las recomendaciones de detección dependen de:

- Edad
- Sexo asignado al nacer
- Antecedentes familiares
- Historial de salud personal
- Factores de estilo de vida (tabaquismo, peso, etc.)

Las guías siguientes provienen del Grupo de Trabajo de Servicios Preventivos de EE. UU. (USPSTF). **Siempre habla con tu médico** — puede recomendar un calendario distinto según tu situación.

## De 18 a 39 años

| Prueba de detección        | Frecuencia          | Notas                                                        |
| -------------------------- | ------------------- | ------------------------------------------------------------ |
| Presión arterial           | Cada 1-2 años       | Más seguido si está alta                                     |
| Colesterol                 | Cada 4-6 años       | Desde los 20 si hay factores de riesgo                       |
| Revisión de cáncer de piel | Habla con tu médico | Especialmente si tienes piel clara o antecedentes familiares |
| Detección de ITS           | Según recomendación | Según actividad sexual                                       |
| Examen dental              | Cada 6 meses        |                                                              |
| Examen de la vista         | Cada 2-4 años       |                                                              |

:::info
Las mujeres también deben hablar sobre: prueba de Papanicolaou (cáncer de cuello uterino) a partir de los 21, examen clínico de mama y detección de clamidia/gonorrea si son sexualmente activas.
:::

## De 40 a 49 años

| Prueba de detección        | Frecuencia               | Notas                                      |
| -------------------------- | ------------------------ | ------------------------------------------ |
| Presión arterial           | Cada año                 |                                            |
| Colesterol                 | Cada 4-6 años            | Más seguido si hay factores de riesgo      |
| Detección de diabetes      | Cada 3 años              | Desde los 35 si hay sobrepeso              |
| Cáncer colorrectal         | **Empezar a los 45**     | Colonoscopia cada 10 años u otras pruebas  |
| Mamografía (mujeres)       | Cada 1-2 años            | Habla con tu médico sobre empezar a los 40 |
| Revisión de cáncer de piel | Anual si hay alto riesgo |                                            |
| Examen dental              | Cada 6 meses             |                                            |
| Examen de la vista         | Cada 2-4 años            |                                            |

:::warning
La detección de cáncer colorrectal ahora empieza a los **45 años** (cambió de 50 en 2021). No la pospongas — la detección temprana salva vidas.
:::

## De 50 a 64 años

| Prueba de detección            | Frecuencia                                 | Notas                                                                                                            |
| ------------------------------ | ------------------------------------------ | ---------------------------------------------------------------------------------------------------------------- |
| Presión arterial               | Cada año                                   |                                                                                                                  |
| Colesterol                     | Cada 4-6 años                              |                                                                                                                  |
| Diabetes                       | Cada 3 años                                |                                                                                                                  |
| Cáncer colorrectal             | Cada 10 años (colonoscopia)                | O prueba FIT anual                                                                                               |
| Mamografía (mujeres)           | Cada 1-2 años                              |                                                                                                                  |
| Cáncer de pulmón (TAC)         | Anual                                      | De 50 a 80 con historial de 20 paquetes-año de tabaco, fumando actualmente o haber dejado en los últimos 15 años |
| Osteoporosis (mujeres)         | A los 65 o antes si hay factores de riesgo | Densitometría ósea                                                                                               |
| Cáncer de próstata (hombres)   | Habla con tu médico                        | De 55 a 69, decisión individual                                                                                  |
| Vacuna contra el herpes zóster | Dos dosis                                  | A partir de los 50                                                                                               |
| Examen dental                  | Cada 6 meses                               |                                                                                                                  |
| Examen de la vista             | Cada 1-2 años                              |                                                                                                                  |

## De 65 años en adelante

| Prueba de detección            | Frecuencia      | Notas                                                                                                                      |
| ------------------------------ | --------------- | -------------------------------------------------------------------------------------------------------------------------- |
| Presión arterial               | Cada año        |                                                                                                                            |
| Colesterol                     | Cada 4-6 años   |                                                                                                                            |
| Cáncer colorrectal             | Hasta los 75    | Hablar después de los 75                                                                                                   |
| Mamografía (mujeres)           | Cada 1-2 años   | Hasta los 74, luego hablar con el médico                                                                                   |
| Cáncer de pulmón (TAC)         | Anual           | Hasta los 80 si fuma actualmente o dejó hace menos de 15 años con 20 paquetes-año                                          |
| Osteoporosis                   | Cada 2 años     |                                                                                                                            |
| Evaluación de riesgo de caídas | Anual           |                                                                                                                            |
| Detección cognitiva            | Anual           |                                                                                                                            |
| Vacuna contra la neumonía      | Una o dos dosis | A partir de los 65, o antes con factores de riesgo (pregunte a su médico o farmacéutico qué vacuna específica le conviene) |
| Vacuna contra la gripe         | Cada año        |                                                                                                                            |
| Examen dental                  | Cada 6 meses    |                                                                                                                            |
| Examen de la vista             | Cada año        |                                                                                                                            |

## Cómo prepararte para las pruebas de detección

1. **Conoce tus antecedentes familiares** — anota qué condiciones hay en tu familia
2. **Lleva tus registros** — resultados previos de detección, registros de vacunas
3. **Pregunta sobre el ayuno** — algunas pruebas de sangre requieren 8-12 horas sin comer
4. **Usa ropa cómoda** — fácil de quitar para los exámenes
5. **Escribe tus preguntas** — no confíes solo en la memoria

:::success
Pon recordatorios de detección en tu calendario. Trátalos como cualquier otra cita importante.
:::
